# Project screenshots

Drop project screenshots here. Reference them from `components/projects/project-data.ts` like:

```ts
screenshots: [
  { src: "/screenshots/regu-dashboard.png", alt: "REGU dashboard" },
  { src: "/screenshots/regu-report.png", alt: "REGU compliance report" },
],
```

**Recommended specs**

- Format: `.png` or `.webp`
- Resolution: max 1600px on the long edge
- Two-column grid at desktop, so 16:10 or 4:3 aspect ratios read best
